HTML Attributları

HTML attributları elementlərə əlavə xüsusiyyətlər verir. Elementlərə müxtəlif xüsusiyyətlər vermək üçün (rəngini, ölçüsünü, görünüşünü, arxa fonunu və s.) start (başlanğıc) tagının içinə bu parametrlər əlavə olunur. Bular attribute adlarnır.

Attributlar həmişə start taglarının içərisində qeyd olunur və ancaq o elementə mənsub olan xüsusiyyətləri özündə əks etdirir.
